FMQB ClickHere Player Generator

Created by Jared i.Greene

Directions:

  • After the song has been cut into 30 seconds, export it as both .M4A and .OGG formats
    • The files must be saved in the following filename format: Artist_Track_30.ogg/m4a
  • Upload the files to the FMQBproductions server in the folder /jPlayer/audio
  • When entering the info into the text fields in this applet, you must truncate the _30 and the middle underscore from the file name, leaving only ArtistName and TrackName within each field
    • Press the Set Artist and Set Track buttons to generate the code
  • Press the "Save To Disk" button, save the ArtistTrack_player.html file to your computer
    • Upload the ArtistTrack_player.html file to the /jPlayer root folder
  • Copy all the code from the text-box labeled "Code For ClickHere popup to player"
    • Paste the code into the source code of the respective AFA page next to Artist - "Track" (Label)
